# Evaluating and signalling trustworthiness in online media

## Project Aim
  This project contains up to two parts, both aimed at supporting users of an online image 
  repository to understand the trustworthiness (or otherwise) of the photographs they 
  view. Part A: using the JPEG Trust framework, create a simple website that ingests and 
  displays uploaded images together with an evaluation (Trust Report) based on 
  individualised Trust Profiles. Part B: develop an intelligent emblem device automatically 
  generated from the Trust Report to denote trustworthiness, displayed adjacent to the 
  image, helping people quickly understand what type of image it is and whether it is 
  what it purports to be (the emblem must be quickly understandable because people 
  view images briefly).

## Expected deliverables
  • Scope is negotiable and will be agreed at the first meeting.
  • Minimum deliverable: define an intelligent emblem generation system (Part B) 
  based on image type, metadata, and provenance:
    o report describing methodology and underlying logic
    o live demonstration
  • Maximum deliverable (if in scope): minimal working website incorporating trust 
    evaluation (Part A).
  • Depending on scope: conceptual skills; web design/development 
    (HTML/CSS/JavaScript); possibly Python; YAML/JSON
## Required Infrastructure
  • Collection of images provided
  • Depending on scope: access to testbed website and associated systems
